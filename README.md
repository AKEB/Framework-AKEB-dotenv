# Dotenv

## Install

composer project akeb/dotenv

Composer config

```json
{
    "require": {
        "akeb/dotenv": "^1.1.0"
    },
    "repositories": [
        {
            "type": "vcs",
            "url": "https://github.com/AKEB/Framework-AKEB-dotenv"
        }
    ]
}
```

or

```bash
composer require akeb/dotenv
```
